学習者向けの実行ビュー（Player）を配置します。`index.html` は MVP のトップ画面です。

## プロジェクト管理機能

### カード式プロジェクト一覧UI

`index.html` には、保存されたプロジェクトを一覧表示するカード式UIが実装されています。

#### 機能

1. **プロジェクト一覧表示**
   - localStorage に保存されたプロジェクトをカード形式で表示
   - 各カードにはプロジェクト名、ファイル名、更新日時が表示される
   - グリッドレイアウトで自動的に配置される

2. **プロジェクト開始**
   - カードの「▶ 開く」ボタンをクリックすると、そのプロジェクトを開始
   - `window.currentProjectData` にプロジェクトデータを設定
   - `demo.html` に遷移してクイズを開始

3. **プロジェクト削除**
   - カードの「🗑 削除」ボタンをクリックすると、確認ダイアログが表示される
   - 確認後、localStorage からプロジェクトを削除

4. **JSONインポート**
   - ファイル入力欄から JSON ファイルを選択してインポート
   - インポートされたプロジェクトは自動的に localStorage に保存される
   - 同名ファイルがある場合は上書きされる

#### データ構造

localStorage の `projects` キーに以下の形式で保存されます：

```json
[
  {
    "name": "プロジェクト名",
    "filename": "project.json",
    "updated_at": "2025-11-21T12:00:00.000Z",
    "data": { /* プロジェクトデータ */ }
  }
]
```

#### エディタとの連携

エディタで `saveProjectAs()` を使用してプロジェクトを保存すると、自動的に localStorage に登録されます。これにより、プレイヤー側で即座にプロジェクト一覧に表示されます。

#### 使用方法

1. エディタでプロジェクトを作成・編集
2. 「💾 プロジェクトを保存」ボタンをクリック
3. ファイル名を入力して保存
4. プレイヤー側の `index.html` を開くと、保存したプロジェクトが一覧に表示される
5. カードの「▶ 開く」ボタンをクリックしてプロジェクトを開始

#### 注意事項

- localStorage はブラウザごとに独立しています
- ブラウザのデータをクリアすると、保存されたプロジェクトも削除されます
- プロジェクトデータは JSON ファイルとしてもダウンロードされるため、バックアップを取ることができます
